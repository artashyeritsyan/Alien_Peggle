# Aliens Peggle

This project is a 2D arcade-style game inspired by classic Peggle mechanics. You control a shooter and launch balls to destroy pegs across different levels using physics and strategy.

## How to Play

1. **Controls:** Use the mouse to aim and click to shoot the ball.  
2. **Shooting Mechanics:** The ball bounces off walls and objects, hitting pegs along its path.  
3. **Peg Clearing:** Destroy pegs to clear the level. Some pegs may have special effects.  
4. **Level Progression:** Each level has a unique layout and increasing difficulty.  
5. **Score Tracking:** The game tracks your shots and time — fewer shots and faster completion lead to better results.  
6. **Game Over:** The game ends when you run out of balls before clearing all required pegs.
